I was checking out an Alfred Unsplash workflow. One thing led to another, and I came upon the section on the author's website which lists the tools he uses for development. I've been meaning to do the same for a while now, but I wasn't sure where I would put it. His page gave me just the kind of inspiration I needed to do so.

Here's my list of my development tools. These may change from time to time, but at least some of them will come in handy for the long haul.

Hardware​

  • Personal Laptop: Macbook Pro 13-inch, Personal Computer: iMac 24-inch
  • Apple Magic Keyboards - 1 for work and 1 for my personal computer
  • Apple Magic Mouse - 1 for work and 1 for my personal computer
  • Logitech M331 Silent Plus Wireless Mouse for personal use
  • Logitech M590 Multi-Device Wireless Mouse for work
  • Dell Pro Stereo Headset UC350 for work
  • Dell UltraSharp 24 FHD Monitor for work
  • Amazon Kindle Paperwhite for all my e-books

Code Editor​

  • GitHub Codespaces - This way I can use Visual Studio Code (along with GitHub's computing resources) right from my browser. I was luckly to be able to get this without subscribing for GitHub Team account or GitHub enterprise, as I had signed up for their beta. If you want to use Visual Studio Code on your browser, you can try out either github1s, vscode.dev or Gitpod

    • GitHub Dark Default Theme
    • My favourite extensions:
      • Wakatime - I use this to track my time spent on coding
      • GitLens - I use this to get more out of Git, such as viewing Git blame annotations
      • Prettier - I use this to format code, especially for my JavaScript projects
      • Tabnine AI Autocompletion and GitHub Copilot - I uses these for code auto completion

  • Jetbrains Toolbox at work - This allows me to easily install and uninstall new versions of Jetbrains IDEs such as PyCharm and IntelliJ Idea

    • Material Theme UI Lite
    • Atom Material Icons
    • My favourite plugins:
      • Git Toolbox

Terminal​

  • Integrated Terminal in Visual Studio Code and in Jetbrains IDEs
  • iTerm2 - I use this in place of the actual Mac terminal as it renders the Powerlevel10k Oh My ZSH Theme more faithfully.

Audio​

  • Spotify
  • FreeYourMusic - This app gives you the flexiblity of switching between different music streaming services as you can easily port over your playlists from one service to another.
  • Dell Pro Stereo Headset UC350 for work
  • Bose QuietComfort 45 Noise Cancelling Headphones for personal use

Other Software & Services​

  • Homebrew - A one-stop shop for installing third-party Mac apps
  • Alfred with Powerpack
  • NordPass as a password manager and NordVPN as a VPN Proxy
  • AVG Antivirus and AVG TuneUp
  • Figma for the design work
  • Speediness to check internet speeds in lieu of sites like Fast and Internet Speed Test by Cloudflare
  • Coconut Battery - This app offers more granular information about your battery such as temperature and cycle count as compared to Mac's native Battery Preferences Pane.
  • Calibre for maintaining my digital library and ease of converting e-Books to different formats like MOBI and AZW3 as well as porting them over to my Kindle
  • Fantastical - It is a great alternative to the Native Mac calendar app that integrates well with Alfred

Everyday Carry​

  • Samsung Galaxy A22 5G
  • Apple Watch Series 6 with a Watch cover
  • Black Teal coloured Crossing Modish Bi Fold Leather Wallet with Flap and Coin Pouch (RFID Protected)

Miscellaneous​
